# mally-test-Markowitz-GA

This program creates optimal portfolio(allocate assets) using genetic algorithm to find the most attractive Risk/Return ratio by Markowitz model

clone
```bash
$ git clone <repo>
```

compile
```bash
$ make
```

run
```bash
./alloc
```
